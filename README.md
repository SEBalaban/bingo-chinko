# Bingo Pachinko

Mobile-first portrait React game:
- Bingo card on top
- Pachinko board on bottom using lightweight physics
- @dnd-kit for positioning the ball at the top
- 5 balls per game, auto-calls BINGO

## Quick start

```bash
npm install
npm run dev
```

Open the dev server URL shown in the terminal (default http://localhost:5173).

## Git setup

```bash
git init
git add .
git commit -m "Initial Bingo Pachinko prototype"
```

## Notes

- TailwindCSS is included for styling (already wired).
- This is an arcade physics prototype (no external physics engine).
